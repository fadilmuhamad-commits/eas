## Getting Started

Step 1 : Install dependencies

```
composer install
npm install
```

Step 2 : Generate key

```
php artisan key:generate
```

Step 3 : Run vite

```
npm run dev
```

Step 4 : Run laravel

```
php artisan serve
```
